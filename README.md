# Case-Social-Connect
Repositório que servirá como 1º Checkpoint da disciplina de WebDev da FIAP

Social Connect:

Projeto de uma página institucional para a empresa fictícia Social Connect, focada no setor automotivo com proposta de carros elétricos personalizados.

Sobre o Projeto:

Este projeto refere-se ao primeiro Checpoint da disciplina Web Development.
O Social Connect é um site estático desenvolvido para apresentar uma empresa inovadora do ramo automotivo.
A página contém seções institucionais como:

Sobre a empresa:
Apresentação da equipe
Informações de contato

O objetivo é oferecer uma interface simples, moderna e responsiva para apresentação da marca.

Tecnologias Utilizadas:
HTML5 – Estrutura da aplicação
CSS3 – Estilização da interface
Flexbox – Organização de layout
Design Responsivo – Adaptação para dispositivos móveis

O projeto possui:

Header com logo e menu de navegação
Seções informativas centralizadas
Imagens ilustrativas
Footer fixo com redes sociais e direitos autorais
